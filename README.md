# Student-Management-System-
This project is a console-based Student Management System developed using Java. It allows users to manage student records efficiently through a menu-driven interface. The system demonstrates Object-Oriented Programming (OOP) concepts such as classes, objects, and encapsulation.

Key functionalities include:

Add Student – Add new student details including ID, Name, Age, and Department.
View Students – Display all student records in a structured tabular format.
Search Student – Search for a student by ID.
Update Student – Modify the details of an existing student.
Delete Student – Remove a student record from the system.

This project is ideal for freshers, college assignments, and technical interview demonstrations, highlighting Java fundamentals, array list manipulation, and basic CRUD operations.
